Format (1+3+3+3)

- --- --- ---

1 first sign  (d goes for directory, - for file)

3 first triad        what the owner can do

3 second triad        what the group members can do

3 third triad        what other users can do

Triad format (r w x)

 first character        r: readable adds 4 to its total (binary 100)

 second character        w: writable adds 2 to its total (in binary 010)

 third character        x: executable adds 1 to its total (in binary 001)

 s or t: executable and setuid/setgid/sticky

 S or T: setuid/setgid or sticky, but not executable

Linux system is about files. Everything is file in linux. Even device driver and network socket. That means, obivously, that you’re mainly working with text editors and terminal prompt. Of course you can use GUI for configuring network card, but usually it’s more efficient to just edit the file manually.

Also to be aware of, linux has different desktop systems, which are just different GUI look for same system. And last but not least, packaging (distrubution of programs, utils, and libs) differ and with that also package manager.

Linux provide good connection between users on same machine, or remote machines. All you need is ssh (client+server) and you can change machines, copy files from one computer to another etc.

In everyday life, you login to system, install new programs, manager existing ones, write some files, watch some logs, read mails etc. All you need is terminal, and perhaps a desktop system, if you’re doing developing or web surfing.
